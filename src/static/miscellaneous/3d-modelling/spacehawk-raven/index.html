SpaceHawk Raven

Created using Blender and GIMP as a test asset for the unreleased indie top-down exploration-driven shooter SpaceHawk.

Lessons were learned and applied from the work done on Viper Rally; a rough block-out was created first, with the high-poly created by retopologizing that, and a low-poly then created by retopologizing that.

The UDK materials supported dynamic paint colour and damage, able to show wear as the player took damage.
